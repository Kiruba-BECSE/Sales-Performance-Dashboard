# 📊 Sales Performance Dashboard (Power BI)

## 📌 Overview
This project focuses on analyzing sales data to generate meaningful business insights using Power BI. The dashboard provides a clear view of key performance indicators (KPIs), helping stakeholders make data-driven decisions.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify high-performing regions and products  
- Track revenue trends over time  
- Provide actionable insights for business growth  

---

## 📂 Dataset
**Source:** Sample / simulated sales dataset  

**Contains:**
- Order Date  
- Region  
- Product Category  
- Sales Amount  
- Quantity Sold  
- Customer Segment  

---

## 🛠️ Tools & Technologies
- **Power BI** – Dashboard creation & visualization  
- **Power Query** – Data cleaning and transformation  
- **Excel** – Data preprocessing  

---

## 📊 Key Features
- Interactive dashboard with filters and slicers  
- KPI cards (Total Sales, Revenue, Quantity)  
- Region-wise and category-wise performance analysis  
- Time-series analysis of sales trends  
- Drill-down capability for detailed insights  

---

## 🔍 Insights Generated
- Identified top-performing regions contributing to revenue  
- Observed seasonal trends in sales  
- Highlighted underperforming product categories  
- Enabled comparison across different segments
